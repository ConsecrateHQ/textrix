# Editor Keyboard Shortcuts

| Feature                 | Mac Shortcut                                                    | Windows/Linux Shortcut                          |
| ----------------------- | --------------------------------------------------------------- | ----------------------------------------------- |
| **Text Formatting**     |                                                                 |                                                 |
| **Bold** (**B**)        | <kbd>⌘</kbd> + <kbd>B</kbd>                                     | <kbd>Ctrl</kbd> + <kbd>B</kbd>                  |
| _Italic_ (_I_)          | <kbd>⌘</kbd> + <kbd>I</kbd>                                     | <kbd>Ctrl</kbd> + <kbd>I</kbd>                  |
| Add Link                | <kbd>⌘</kbd> + <kbd>K</kbd>                                     | <kbd>Ctrl</kbd> + <kbd>K</kbd>                  |
| Header (H1)             | <kbd>⌘</kbd> + <kbd>Opt</kbd> + <kbd>1</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>1</kbd> |
| Subheader (H2)          | <kbd>⌘</kbd> + <kbd>Opt</kbd> + <kbd>2</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>2</kbd> |
| Blockquote              | <kbd>⌘</kbd> + <kbd>Opt</kbd> + <kbd>5</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>5</kbd> |
| Horizontal Line         | <kbd>⌘</kbd> + <kbd>Enter</kbd>                                 | <kbd>Ctrl</kbd> + <kbd>Enter</kbd>              |
| Code Block              | <kbd>⌘</kbd> + <kbd>Opt</kbd> + <kbd>6</kbd>                    | <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>6</kbd> |
| Line Break              | <kbd>Shift</kbd> + <kbd>Enter</kbd>                             | <kbd>Shift</kbd> + <kbd>Enter</kbd>             |
| **Lists & Indentation** |                                                                 |                                                 |
| Outdent List            | <kbd>⌘</kbd> + <kbd>[</kbd> / <kbd>Shift</kbd> + <kbd>Tab</kbd> | <kbd>Shift</kbd> + <kbd>Tab</kbd>               |
| Indent List             | <kbd>⌘</kbd> + <kbd>]</kbd> / <kbd>Tab</kbd>                    | <kbd>Tab</kbd>                                  |

### Editor Editing Shortcuts

| Feature      | Syntax                                    |
| ------------ | ----------------------------------------- |
| Ordered List | Start with `1. ` (or any number + dot)    |
| Bullet List  | Start with `* ` (asterisk + space)        |
| Code Block   | Use triple backticks ` ``` `              |
| Inline Code  | Wrap text with backticks (e.g., \`code\`) |
| Emoji Picker | Type `:` then search for an emoji         |

### Markdown Shortcuts (Requires `markdownShortcuts` options to be enabled)

| Feature             | Syntax Example                               |
| ------------------- | -------------------------------------------- |
| Blockquote          | Type `>` for blockquote, `>>` for pull quote |
| **Bold** Text       | Type `**bold**` or `__bold__`                |
| _Italic_ Text       | Type `_italic_` or `*italic*`                |
| Horizontal Line     | Type `---` for a divider                     |
| # Big Title (H1)    | Type `# Title`                               |
| ## Small Title (H2) | Type `## Subtitle`                           |

**Tip:** These shortcuts also work when pasting formatted text!
